# 距離感知實驗報告 (Distance Perception Experiment Report)

## 1. 實驗目的
驗證 Binaural Speech Synthesis 模型是否具備正確的**距離感知能力**。具體而言，我們希望確認當輸入的 Tx 與 Rx 距離改變時，模型生成的雙耳音訊是否會呈現正確的**音量變化 (Inverse Square Law)** 與**混響變化**。

## 2. 實驗設置
我們基於 `Subject 1` 創建了 5 組測試變體，強制修改其 `tx_positions.txt` 中的距離參數，但保留原始的語音內容 (`mono.wav`) 和參考音訊 (`binaural.wav`，皆為 100cm 版本)。

*   **變數**: 聲源距離 (0.2m, 0.5m, 1.0m, 2.0m, 3.0m)
*   **控制變數**: Tx 方向 (Face Forward), Rx 位置 (Origin), 輸入語音內容。
*   **參考基準 (Reference)**: 原始 100cm 的真實錄音。

## 3. 實驗結果數據

| 距離 (Distance) | L2 誤差 | Amp 誤差 | Phase 誤差 | 解讀 |
| :--- | :--- | :--- | :--- | :--- |
| **20cm** (超近) | 0.068 | 0.021 | 1.568 | ❌ **異常**：誤差極低，幾乎與 100cm 無異。 |
| **50cm** (近) | 0.201 | 0.032 | 1.574 | 誤差略增，有改變但幅度不大。 |
| **100cm** (基準) | **0.068** | **0.020** | **1.584** | ✅ **基準點**：模型表現最佳 (Expected)。 |
| **200cm** (遠) | 0.574 | 0.042 | 1.558 | ✅ **正常**：誤差顯著增加，代表生成音量變小。 |
| **300cm** (超遠) | 0.786 | 0.050 | 1.581 | ✅ **正常**：誤差最大，代表生成音量變得更小。 |

> **註**: 因為 Reference 始終是 100cm 的音量，所以當我們要求模型生成 300cm (應變小聲) 時，與 Reference 的誤差**越大**反而代表模型**做對了**（因為它正確地變得跟 100cm 不一樣）。

## 4. 分析與結論

### 遠場表現 (Far-field Performance) ✅ 
模型在 **> 100cm** 的距離表現符合物理預期。隨著距離增加 (1m -> 2m -> 3m)，Amplitude Error 顯著上升，這證明模型成功學會了**將聲音變小聲**並增加混響，與原始的 100cm 錄音產生了明顯區別。

### 近場表現 (Near-field Performance) ❌
模型在 **< 50cm** (特別是 20cm) 的表現**失效**。理論上 20cm 的聲音應該比 100cm 大聲非常多（根據平反比定律），因此 Amplitude Error 應該要非常巨大。然而數據顯示誤差僅為 0.021 (與 100cm 的 0.020 幾乎相同)。
*   **推論**: 模型可能在訓練資料中缺乏極近距離的樣本，或者模型架構限制了最大輸出的音量，導致它無法正確生成「貼在耳邊」的效果。

## 5. 建議
*   **聽感測試**: 請務必聆聽 `subject1_dist_20cm.wav`，確認它是否真的不夠大聲。
*   **應用限制**: 目前模型適用於 1公尺以外的場景，若應用於 "Whisper" 或 "貼耳" 場景可能效果不彰。
