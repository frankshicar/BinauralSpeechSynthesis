## 📊 重要發現摘要

### 角度估計

1. **ITD 方法遠優於 ILD 方法**
   - ITD (GCC-PHAT) 誤差約 3-4°
   - ILD 線性映射誤差高達 25°
   - 語音信號應該使用 ITD

2. **+60° ~ +90° 區間的非單調性**
   - 模型在此區間存在「先升後降再升」的現象
   - 曲線插值在此區間失效
   - 需要分段校正策略

3. **HRTF 不匹配是根本原因**
   - 預訓練模型的 HRTF 與 KEMAR 不同
   - 右側偏差大於左側
   - 徹底解決需要 fine-tune 模型

### 訓練優化

1. **IPD Loss 可以改善 ITD 準確性**
   - 在頻域直接約束雙耳相位差
   - 比時域 ITD 更精確且可微分
   - 建議權重：0.05 ~ 0.1

2. **訓練記錄功能提供完整追蹤**
   - 自動記錄所有 loss 值和學習率
   - 即時監控和視覺化
   - JSON/CSV 多種格式輸出

### 測試集設計

1. **所有 subject 的 mono.wav 相同是正常的**
   - 這是標準的 HRTF 測試設計
   - 用相同音源測試不同 HRTF

2. **訓練集角度分布合理**
   - 前方密度高（0.42%），後方密度低（0.08%）
   - 所有測試角度都有充足樣本
   - 360° 全方位覆蓋
