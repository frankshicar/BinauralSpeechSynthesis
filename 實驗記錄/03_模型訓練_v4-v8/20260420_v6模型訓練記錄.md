## 二、Loss Function 定義與理論

### 2.1 總 Loss 函數

```python
total_loss = (
    l2_weight * L2Loss(pred, target) +
    phase_weight * PhaseLoss(pred, target) +
    ipd_weight * IPDLoss(pred, target)
)
```

v6 **未使用** WarpLoss 和 SmoothLoss（v5 曾實驗但效果不佳）。

### 2.2 各 Loss 項的數學定義

#### L2 Loss（時域波形重建）

```python
L2Loss = mean((pred - target)²)
```

**物理意義：**
- 最小化預測波形與目標波形的均方誤差
- 優化整體能量和波形形狀
- 對應客觀指標：amplitude error

**優點：**
- 簡單、穩定、易優化
- 保證基本音質

**缺點：**
- 對相位不敏感（相位偏移不影響 L2）
- 無法捕捉空間感知

**參考文獻：**
- Richard et al. (2021) 原始論文使用 L2 作為主要 loss

---

#### Phase Loss（頻域相位一致性）

```python
# STFT 計算
pred_stft = STFT(pred)    # shape: (freq, time, 2_channels)
target_stft = STFT(target)

# 相位提取
pred_phase = atan2(pred_stft.imag, pred_stft.real)
target_phase = atan2(target_stft.imag, target_stft.real)

# Phase Loss（循環距離）
phase_diff = pred_phase - target_phase
phase_diff = atan2(sin(phase_diff), cos(phase_diff))  # wrap to [-π, π]
PhaseLoss = mean(abs(phase_diff))
```

**物理意義：**
- 最小化頻域相位差
- 保留音色和時間結構
- 影響垂直定位（spectral cues）

**為什麼需要 Phase Loss？**
- L2 loss 對相位不敏感：`sin(t)` 和 `sin(t + π/4)` 的 L2 差異很小，但聽感完全不同
- 相位對音色至關重要（formant structure）
- 相位影響 ITD（Interaural Time Difference）

**參考文獻：**
- Rafaely et al. (2025): "Phase loss 影響音色和垂直定位"
- Richard et al. (2021): 原始論文使用 amplitude + phase loss

---

#### IPD Loss（雙耳相位差，空間定位核心）

```python
# 左右耳 STFT
left_stft = STFT(pred[:, 0])   # 左耳
right_stft = STFT(pred[:, 1])  # 右耳

# 計算 IPD（Interaural Phase Difference）
pred_ipd = angle(left_stft) - angle(right_stft)
target_ipd = angle(target_left_stft) - angle(target_right_stft)

# IPD Loss（循環距離）
ipd_diff = pred_ipd - target_ipd
ipd_diff = atan2(sin(ipd_diff), cos(ipd_diff))  # wrap to [-π, π]
IPDLoss = mean(abs(ipd_diff))
```

**物理意義：**
- IPD 直接對應 **ITD（Interaural Time Difference）**
- ITD 是人類水平定位的主要 cue（尤其低頻 < 1.5 kHz）
- IPD = 2πf × ITD

**為什麼 IPD 是空間定位核心？**
- **Duplex Theory of Sound Localization (Rayleigh, 1907):**
  - 低頻（< 1.5 kHz）：主要靠 ITD/IPD
  - 高頻（> 1.5 kHz）：主要靠 ILD（Interaural Level Difference）
- IPD 包含了左右耳的時間差信息
- 人類對 IPD 的 JND（Just Noticeable Difference）約 10-20 μs，對應 1-2° 的角度分辨率

**參考文獻：**
- Rafaely et al. (2025): "IPD 直接對應水平定位"
- Tokala et al. (2024): 顯式在 loss 中加入 IPD 項改善定位
- Blauert (1997): "Spatial Hearing" - ITD/ILD 的經典理論

### 2.3 未使用的 Loss 項（v5 實驗）

#### WarpLoss（幾何先驗約束）

```python
WarpLoss = mean((neural_warp - geometric_warp)²)
```

**理論：**
- 讓神經網路學到的 warp field 接近幾何計算的 warp
- 用物理先驗約束網路，減少訓練資料需求

**v5 實驗結果：**
- 權重 λ_warp = 0.01
- **效果不明顯**，可能原因：
  - 幾何 warp 過於簡化（僅考慮距離，忽略 HRTF）
  - 約束過強反而限制網路學習能力
- v6 移除此項

**參考文獻：**
- Physics-informed neural networks (PINNs) 的相關工作
- 用物理約束輔助深度學習

---

#### SmoothLoss（時間平滑性）

```python
# Warp field 的時間梯度
warp_gradient = warpfield[:, 1:] - warpfield[:, :-1]
SmoothLoss = mean(abs(warp_gradient))
```

**理論：**
- 懲罰 warp field 的劇烈跳變
- 確保說話者移動時空間位置變化平滑

**v5 實驗結果：**
- 權重 λ_smooth = 0.001
- **效果不明顯**，可能原因：
  - 訓練資料中說話者移動速度本就平滑
  - WaveNet 的 receptive field 已提供時間平滑性
- v6 移除此項

**參考文獻：**
- Total Variation (TV) regularization
- 常用於影像去噪和訊號平滑

### 2.4 Loss 數值範圍與權重設計

**實測數值範圍（v6 epoch 80）：**

| Loss | 數值範圍 | 量級 |
|------|---------|------|
| L2 | 0.00009 | 10⁻⁴ |
| Phase | 0.97 | 10⁰ |
| IPD | 0.88 | 10⁰ |

**數值範圍差異達 10,000 倍！**

**權重設計原則（基於 Kendall et al. 2018）：**

```
實際貢獻 = 權重 × loss 數值

目標：讓各 loss 的實際貢獻在同一量級
```

**v6 的權重選擇：**

| Stage | L2 權重 | Phase 權重 | IPD 權重 | 設計理念 |
|-------|---------|-----------|---------|---------|
| 1 | 10.0 | 0.0 | 0.0 | 純 L2，打好基礎 |
| 2 | 10.0 | 0.1 | 0.2 | IPD 主導（64%），Phase 輔助（36%）|
| 3 | 10.0 | 0.15 | 0.3 | 提高 Phase/IPD，加速空間學習 |

**實際貢獻計算（Stage 2）：**
```
L2:    10.0 × 0.00009 = 0.0009  (0.3%)
Phase:  0.1 × 0.97    = 0.097   (36%)
IPD:    0.2 × 0.88    = 0.176   (64%)
Total: 0.274
```

**觀察：** Phase/IPD 已主導訓練（99.7%），但改善速度仍慢，可能需要更大的權重。

## 三、三階段訓練策略

### 設計原則與理論依據

#### 1. Curriculum Learning（課程學習）

**理論基礎：**
- Bengio et al. (2009). "Curriculum learning." ICML.
- 核心概念：先訓練簡單任務，再逐步增加難度，可加速收斂並提升最終性能

**應用於本研究：**
- Stage 1：純 L2 loss（波形重建）→ 最基礎的任務
- Stage 2：加入 Phase + IPD → 引入空間感知
- Stage 3：提高 Phase/IPD 權重 → 精修空間定位

#### 2. Multi-Task Learning Loss Balancing（多任務學習損失平衡）

**核心挑戰：** L2、Phase、IPD 三個 loss 的數值範圍差異極大：
- L2 loss ≈ 0.0001（小）
- Phase loss ≈ 0.8-1.0（大）
- IPD loss ≈ 0.6-0.9（大）

**參考論文與方法：**

##### (1) Uncertainty-based Weighting
- **Kendall et al. (2018). "Multi-Task Learning Using Uncertainty to Weigh Losses for Scene Geometry and Semantics." CVPR.**
- 理論：用可學習的不確定性參數自動平衡不同尺度的 loss
- 公式：`L_total = (1/σ₁²)·L₁ + (1/σ₂²)·L₂ + log(σ₁) + log(σ₂)`
- 本研究未採用原因：需要額外的可學習參數，增加訓練複雜度

##### (2) GradNorm
- **Chen et al. (2018). "GradNorm: Gradient Normalization for Adaptive Loss Balancing in Deep Multitask Networks." ICML.**
- 理論：動態調整權重使各任務的梯度範數相近
- 本研究未採用原因：需要在訓練過程中動態調整權重，實作複雜

##### (3) Fixed Weighting with Curriculum（本研究採用）
- 結合固定權重 + 階段性引入任務
- 優點：簡單、可控、易於調試
- 權重設計原則：**實際貢獻 = 權重 × loss 數值**

#### 3. 音訊領域的 Loss Weighting 實踐

**參考文獻：**
- **Rafaely et al. (2025). "Loss functions incorporating auditory spatial perception in deep learning – a review." I3DA 2025 / arXiv:2506.19404.**
  - 注意：此論文於 2025 年 6 月 24 日提交 arXiv，發表於 I3DA 2025 會議（DOI: 10.1109/I3DA65421.2025.11202063）
  - 論文摘要確認：綜述雙耳音訊 loss function 設計，重點在 ITD/ILD 等空間定位 cue
  - 關鍵發現：
    - L2 loss 主要優化波形能量，對空間感貢獻有限
    - IPD (Interaural Phase Difference) 直接對應水平定位
    - Phase loss 影響音色和垂直定位
    - Perceptual loss 權重通常需要是 L2 的 10-100 倍（因數值範圍差異）

- **Tokala et al. (2024). "Binaural speech enhancement using deep complex convolutional transformer networks." ICASSP.**
  - 使用 SNR + STOI + ILD + IPD 的複合 loss
  - 明確在 loss function 中加入 ILD/IPD 保留項
  - 證明顯式的空間 cue loss 可改善定位準確性

#### 4. Learning Rate Scheduling in Multi-Stage Training

**問題：** v5 在 Stage 切換時 LR 已衰減過度，新 loss 項無法有效學習

**解決方案：Stage 切換時重置 LR**
- **Smith (2017). "Cyclical Learning Rates for Training Neural Networks." WACV.**
  - 理論：週期性提高 LR 可幫助跳出局部最優
- **Loshchilov & Hutter (2017). "SGDR: Stochastic Gradient Descent with Warm Restarts." ICLR.**
  - 理論：在訓練過程中重啟 LR 可改善收斂
- 本研究應用：每次 Stage 切換時將 LR 重置為 0.0003，確保新 loss 項有足夠的學習率

### 各階段配置

| 階段 | Epochs | L2 | Phase | IPD | 目標 | 實際貢獻比例* |
|------|--------|----|-------|-----|------|--------------|
| Stage 1 | 0–80 | **10.0** | 0.0 | 0.0 | 純波形重建，打好 L2 基礎 | L2: 100% |
| Stage 2 | 80–140 | **10.0** | 0.1 | 0.2 | 加入空間感知 | L2: 0.4%, Phase: 36%, IPD: 64% |
| Stage 3 | 140–200 | **10.0** | 0.15 | 0.3 | 精修空間定位 | L2: 0.4%, Phase: 54%, IPD: 108% |

*實際貢獻 = 權重 × loss 數值（以 epoch 80 數值估算：L2≈0.00009, Phase≈0.97, IPD≈0.88）

### 權重設計的考量

#### Stage 1: 純 L2 訓練
- **目標：** 讓網路學會基本的 mono → binaural 映射
- **L2 = 10.0：** 足夠大以主導訓練，但不會過大導致過擬合
- **Phase/IPD = 0.0：** 完全關閉，避免干擾 L2 收斂

#### Stage 2: 引入空間感知
- **L2 = 10.0（維持）：** 保持波形品質，但實際貢獻僅 0.4%
- **Phase = 0.1：** 實際貢獻 ≈ 0.097（36%），開始學習相位
- **IPD = 0.2：** 實際貢獻 ≈ 0.176（64%），IPD 為主要優化目標
- **設計理念：** IPD 是水平定位的核心 cue，給予最高權重

#### Stage 3: 精修
- **Phase/IPD 權重各提高 50%：** 加速空間感知的學習
- **L2 維持不變：** 作為品質保底

### 已知限制與未來改進方向

**當前權重的問題（2026-04-21 分析）：**
- Stage 2/3 的 Phase/IPD 權重可能過於保守
- 從 epoch 103-140 的訓練 log 顯示：
  - Phase 僅從 0.831 降至 0.812（-2.3%）
  - IPD 僅從 0.651 降至 0.639（-1.8%）
  - 改善速度過慢

**理論建議的權重範圍（基於 Rafaely et al. 2025）：**
```python
# 更激進的配置（未測試）
"stage2": {"l2": 10.0, "phase": 0.5, "ipd": 1.0}   # Phase/IPD 提高 5 倍

```

**未來可嘗試的方法：**
1. **Uncertainty-based weighting：** 讓網路自動學習最佳權重
2. **GradNorm：** 動態平衡梯度範數
3. **更激進的 Phase/IPD 權重：** 基於當前訓練結果，0.1/0.2 可能太小

## 四、與前版本的差異

### v6 vs v5 的關鍵改變

**1. L2 權重全程保持 10.0（最重要的改變）**

v5 的 L2 權重在 Stage 1 就降到 3.0，Stage 3 更降到 2.0，導致波形重建基礎不夠紮實。
v6 全程維持 10.0，確保模型在學習空間感知的同時不犧牲波形品質。

**2. Stage 1 從 50 epochs 延長至 80 epochs**

v5 的 Stage 1 在 epoch 50 結束時 L2 仍在持續下降（0.000555 → 0.000164），尚未收斂就切換。
v6 給 80 epochs 讓 L2 充分收斂，再引入 Phase/IPD。

**3. Stage 切換時重置 LR（新增機制）**

v5 的問題：Stage 2 在 epoch 51 加入 IPD 後，accumulated_loss 從 0.178 暴增到 0.674，
ReduceLROnPlateau 的 patience 計時器重置，導致 LR 在 Stage 2 才進行 16 個 epoch 就衰減（epoch 66），
Stage 3 開始前 LR 已降至 0.000125，新的 loss 訊號無法有效學習。

v6 在每次 Stage 切換時：
- 將 LR 重置為 0.0003（初始值的 30%）
- 重置 scheduler 的 best 值和 patience 計數器

**4. LR patience 從 15 提高至 20**

配合更長的 Stage 1，避免純 L2 訓練期間 LR 過早衰減。

理由：v5/v6 的 Phase 權重較高

**IPD / 角度誤差:**
```
v6 ≈ v5 > v4 ≈ Meta
```
理由：v6/v5 都有顯式的 IPD loss

**整體品質:**
```
v6 > v5 ≈ Meta > v4
```
理由：v6 在波形品質和空間感知間取得最佳平衡

## 五、預期改善

根據 v5 的問題分析，v6 預期：

- **L2 / Amplitude**：因 L2 權重全程 10.0 + Stage 1 充分收斂，應追平或超越 Meta pretrained
- **Phase / IPD**：Stage 切換 LR 重置確保 Stage 2/3 有足夠學習能力，維持 v5 的相位優勢
- **角度誤差**：+60° 的系統性誤差（13°）受限於訓練資料左右不對稱，預期改善有限

## 六、已知限制

- 訓練資料（Meta dataset）前方角度樣本比側方多約 60%，導致 ±60° 誤差系統性偏大
- 資料為固定場景錄製，無法增加，weighted sampling 為可能的補救方向
- +60° 誤差在 v4/v5/Meta 三個模型都是 13°，屬於資料層面的限制

### 7.4 重要發現

**1. v6 在 Meta 官方 testset 上的表現超越 Meta pretrained（除 L2 外）**

這是本系列實驗最重要的突破。Amplitude、Phase、ITD、ILD 全面改善，代表：
- L2 全程高權重（10.0）確保了波形品質基礎
- IPD loss 有效改善了雙耳時間差的精確度
- Stage 切換 LR 重置機制讓 Stage 2/3 的學習更有效

**2. 角度誤差幾乎追平 Meta pretrained**

v6 平均角度誤差 4.3° vs Meta 的 4.2°，差距僅 0.1°，在誤差範圍內可視為相當。
考慮到 v6 使用的訓練資料包含靜態角度（與 Meta 的動態資料不同分布），這個結果相當優秀。

**3. +60° 誤差首次低於 Meta pretrained**

subject6（+60°）：v6 = 10.1° vs Meta = 13.0°，改善 22%。
這是歷代模型中首次在 +60° 超越 Meta，可能得益於 IPD loss 對側方角度的強化學習。

**4. 訓練資料分布問題仍存在**

subject2（-60°）誤差 7.1°，比 Meta 的 0.2° 差很多，顯示左側 60° 仍有系統性問題。
這與訓練資料左右不對稱有關（左側樣本略多於右側），但 +60° 的改善說明 IPD loss 有部分補救效果。

## 九、訓練日誌與分析

### 2026-04-21 中期分析（Epoch 142）

**訓練進度：**
- 已完成 Stage 1 (0-80) 和 Stage 2 (80-140)
- 正在進行 Stage 3 (140-200)，當前 epoch 142

**Stage 2 表現（Epoch 80-140）：**
- L2: 0.0000886 → 0.000110 (+24%) - 輕微反彈後穩定
- Phase: 0.969 → 0.812 (-16%) - 持續改善
- IPD: 0.876 → 0.639 (-27%) - 顯著改善

**觀察與問題：**
1. **L2 反彈是正常的：** 加入新 loss 項後，網路需要重新平衡內部表徵
2. **Phase/IPD 改善速度偏慢：** 60 個 epoch 僅改善 16-27%
3. **權重可能過於保守：** Phase=0.1, IPD=0.2 的實際貢獻遠小於 L2

**理論分析（基於 Kendall et al. 2018）：**
```
實際貢獻 = 權重 × loss 數值

Stage 2 (epoch 80):
- L2:    10.0 × 0.0000886 = 0.000886 (0.3%)
- Phase:  0.1 × 0.969     = 0.0969   (36%)
- IPD:    0.2 × 0.876     = 0.1752   (64%)
```

**結論：** Phase/IPD 已經主導訓練（佔 99.7%），但改善速度仍慢，可能是：
1. 權重絕對值太小，梯度訊號不夠強
2. 需要更長的訓練時間
3. 網路容量限制（blocks=3 可能不足）

**下一步：**
- 觀察 Stage 3 (epoch 140-200) 的效果
- 如果 Phase/IPD 仍無顯著改善，考慮：
  - 提高 Phase/IPD 權重至 0.5/1.0
  - 或增加 WaveNet blocks 至 4-5
