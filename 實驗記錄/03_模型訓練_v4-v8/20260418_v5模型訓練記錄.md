### 三階段訓練策略

#### Stage 1: 音質優先階段（Epoch 0-50）
```json

{
  "l2": 2.0,      // 降低音質權重
```

## 七、訓練策略說明

### 為什麼採用三階段訓練？

1. **避免多目標衝突**
   - L2 Loss 和 IPD Loss 可能存在衝突：
     - L2 要求整體波形相似
     - IPD 要求左右耳相位差準確
   - 同時優化可能導致模型在兩者之間搖擺不定

2. **漸進式學習**
   - Stage 1：先建立穩固的音質基礎
   - Stage 2：在音質基礎上引入空間定位
   - Stage 3：在兩者之間找到最佳平衡

3. **參考人類學習過程**
   - 類似於先學會「說話」，再學會「在正確的位置說話」

## 九、技術細節

### Loss Function 組成

```python
total_loss = (
    l2_weight * L2Loss(pred, target) +
    phase_weight * PhaseLoss(pred, target) +
    ipd_weight * IPDLoss(pred, target) +
    lambda_warp * WarpLoss(neural_warp, geometric_warp) +
    lambda_smooth * SmoothLoss(warpfield)
)
```

### IPD Loss 說明
- **IPD (Interaural Phase Difference)**：雙耳相位差
- 直接優化左右耳之間的相位差，而不是單獨優化每個耳朵的相位
- 對空間定位（角度準確性）有直接影響

### Warp Loss 說明
- **Neural Warp**：神經網路學習的 warp field
- **Geometric Warp**：基於幾何計算的 warp field
- Warp Loss 懲罰兩者之間的偏差，確保模型學習到物理上合理的空間變換

**新增的空間感知損失**：

1. **IPD Loss (Interaural Phase Difference)**
   ```python
   class IPDLoss(Loss):
       """計算左右耳相位差的誤差"""
       def _loss(self, data, target):
           # 計算左右耳的相位
           left_phase = atan2(left_stft)
           right_phase = atan2(right_stft)
           
           # 計算 IPD
           pred_ipd = left_phase - right_phase
           target_ipd = target_left_phase - target_right_phase
           
           # 處理週期性
           ipd_diff = abs(pred_ipd - target_ipd)
           ipd_diff = π - abs(ipd_diff - π)
           
           return mean(ipd_diff)
   ```
   
   **作用**：
   - 直接優化雙耳相位差，改善水平定位
   - 對 ITD 有間接影響
   - 屬於 SL-NN (Spatial Loss for Neural Networks) 類別

2. **ITD Loss (Interaural Time Difference)** - 已實作但未使用
   ```python
   class ITDLoss(Loss):
       """計算左右耳時間差的誤差"""
       def _compute_itd(self, stereo_signal):
           # 使用互相關方法計算 ITD
           correlations = []
           for shift in range(-max_shift, max_shift+1):
               corr = sum(left[shift:] * right[:-shift])
               correlations.append(corr)
           
           # 找到最大相關對應的延遲
           itd_samples = argmax(correlations) - max_shift
           return itd_samples / sample_rate * 1e6  # 轉換為微秒
   ```
   
   **未使用原因**：
   - 計算成本高（需要滑動窗口互相關）
   - IPD Loss 已經隱式優化了 ITD
   - 可能在未來版本中啟用

3. **ILD Loss (Interaural Level Difference)** - 已實作但未使用
   ```python
   class ILDLoss(Loss):
       """計算左右耳強度差的誤差"""
       def _compute_ild(self, stereo_signal):
           # 在高頻段計算能量
           left_energy = sum(left_power[1000:8000Hz])
           right_energy = sum(right_power[1000:8000Hz])
           
           # 計算 ILD (dB)
           ild = 10 * log10(left_energy / right_energy)
           return ild
   ```
   
   **未使用原因**：
   - 主要影響高頻定位
   - 語音信號的低頻成分更重要
   - 可能在未來版本中啟用

**GCC-PHAT 角度估計**：
```python
def gcc_phat_estimate(binaural_audio, sample_rate=48000):
    """使用 GCC-PHAT 算法估計方位角"""
    left, right = binaural_audio[0], binaural_audio[1]
    
    # FFT
    left_fft = fft(left)
    right_fft = fft(right)
    
    # 互功率譜 + PHAT 加權
    cross_spectrum = left_fft * conj(right_fft)
    gcc_phat = cross_spectrum / abs(cross_spectrum)
    
    # 反 FFT 得到互相關
    correlation = ifft(gcc_phat)
    
    # 找到峰值對應的 ITD
    delay_samples = argmax(correlation) - len(correlation)//2
    
    # ITD → 方位角
    azimuth = arcsin(sound_speed * delay_samples / (sample_rate * ear_distance))
    return degrees(azimuth)
```
