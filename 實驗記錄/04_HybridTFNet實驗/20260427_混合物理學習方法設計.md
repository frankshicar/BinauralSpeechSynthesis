### 我們的方法（改進版）
```python
# 我們: Phase-based + 時域優化

# 1. 物理 ITD (低頻精確)
physical_ITD = head_radius × sin(azimuth) / sound_speed
physical_phase = 2π × freq × physical_ITD

# 2. 學習 ILD (magnitude)
ILD = MagnitudeNet(mono_stft, view)

# 3. 學習殘差 ITD (修正物理模型的不足)
residual_ITD = ResidualNet(mono_stft, view)  # 學習頻率相關的修正
total_ITD = physical_ITD + residual_ITD

# 4. 合成
phase_L = phase_mono + π × freq × total_ITD
phase_R = phase_mono - π × freq × total_ITD

Y_L = ILD_L × exp(j × phase_L)
Y_R = ILD_R × exp(j × phase_R)

# 5. 時域 L2 loss
binaural_pred = ISTFT([Y_L, Y_R])
loss = MSE(binaural_pred, binaural_gt)
```

## 詳細設計

### 架構

```python
class HybridPhysicalLearned(nn.Module):
    def __init__(self):
        # 1. Magnitude 分支 (學習 ILD)
        self.magnitude_net = MagnitudeNet()
        
        # 2. Residual ITD 分支 (學習物理模型的殘差)
        self.residual_itd_net = ResidualITDNet()
        
        # 3. 物理參數
        self.head_radius = 0.0875  # 8.75cm
        self.sound_speed = 343.0   # m/s
    
    def forward(self, mono, view):
        # STFT
        mono_stft = STFT(mono)  # B×F×T
        
        # === Magnitude 分支 ===
        mag_L, mag_R = self.magnitude_net(mono_stft, view)
        
        # === Phase 分支 ===
        # 1. 物理 ITD (幾何計算)
        azimuth = view[:, 0]  # 假設第一維是方位角
        physical_ITD = self.compute_physical_ITD(azimuth)  # B×1
        
        # 2. 學習殘差 ITD (頻率相關)
        residual_ITD = self.residual_itd_net(mono_stft, view)  # B×F
        
        # 3. 總 ITD
        total_ITD = physical_ITD.unsqueeze(1) + residual_ITD  # B×F
        
        # 4. 計算相位差
        freq = torch.linspace(0, 24000, mono_stft.size(1))  # 0-24kHz
        phase_shift = 2 * np.pi * freq * total_ITD  # B×F
        
        # 5. 應用到 mono phase
        phase_mono = torch.angle(mono_stft)
        phase_L = phase_mono + phase_shift / 2
        phase_R = phase_mono - phase_shift / 2
        
        # === 合成 ===
        Y_L = mag_L * torch.exp(1j * phase_L)
        Y_R = mag_R * torch.exp(1j * phase_R)
        
        # ISTFT
        binaural = ISTFT(torch.stack([Y_L, Y_R], dim=1))
        
        return binaural, {
            'mag_L': mag_L,
            'mag_R': mag_R,
            'physical_ITD': physical_ITD,
            'residual_ITD': residual_ITD,
            'total_ITD': total_ITD,
        }
    
    def compute_physical_ITD(self, azimuth):
        """
        幾何 ITD 計算
        ITD = (r/c) × (sin(θ) + θ)  # Woodworth 公式
        """
        theta = azimuth * np.pi / 180  # 轉弧度
        ITD = (self.head_radius / self.sound_speed) * (
            torch.sin(theta) + theta
        )
        return ITD
```

## 為什麼這個方法可行？

### 1. 物理 ITD 提供基礎

**低頻 (<1kHz)**:
```
ITD 主導空間感知
物理模型已經很準確
λ_ITD = 1.0 × ITD_physical
```

**優點**:
- 不需要學習低頻 phase
- 物理保證正確
- 梯度穩定

### 2. 殘差 ITD 修正細節

**中高頻 (1-8kHz)**:
```
物理模型不夠精確（頭部不是完美球體）
學習頻率相關的修正
λ_residual = 0.1 × ITD_residual
```

**優點**:
- 學習空間小（只是修正）
- 有物理先驗約束
- 更容易收斂

### 3. ILD 學習已證明可行

**所有頻率**:
```
Magnitude 可以學習（已驗證）
λ_ILD = 1.0 × ILD_loss
```

**優點**:
- 已經證明可學習
- 提供能量線索
- 與 ITD 互補

### 4. 時域 L2 端到端優化

**最終目標**:
```
loss = MSE(waveform_pred, waveform_gt)
```

**優點**:
- 直接優化聽覺目標
- 避免頻域 phase wrapping
- 端到端可微分

## 訓練策略

### Loss 設計

```python
def compute_loss(pred, target, outputs, config):
    # 1. 時域 L2 (主要目標)
    loss_waveform = MSE(pred, target)
    
    # 2. Magnitude 約束 (輔助)
    pred_stft = STFT(pred)
    target_stft = STFT(target)
    loss_mag = MSE(|pred_stft|, |target_stft|)
    
    # 3. ITD 正則化 (防止過大)
    residual_ITD = outputs['residual_ITD']
    loss_itd_reg = torch.mean(residual_ITD ** 2)
    
    # 4. IPD 監控 (不參與訓練)
    pred_ipd = angle(pred_stft_L / pred_stft_R)
    target_ipd = angle(target_stft_L / target_stft_R)
    loss_ipd = mean(1 - cos(pred_ipd - target_ipd))
    
    # Total
    loss = (
        1.0 * loss_waveform +
        0.5 * loss_mag +
        0.01 * loss_itd_reg
    )
    
    return loss, {
        'waveform': loss_waveform,
        'mag': loss_mag,
        'itd_reg': loss_itd_reg,
        'ipd': loss_ipd,  # 監控
    }
```

## 與 Meta 的差異

### Meta 的優勢
- Warp 是時域操作，更直接
- 不需要 STFT/ISTFT
- 計算效率高

### 我們的優勢
- **顯式 ILD 學習** - Meta 沒有明確處理 magnitude
- **頻率相關 ITD** - 更精細的控制
- **物理 + 學習分離** - 更好的可解釋性
- **STFT 域操作** - 更容易分析和調試

### 創新點

1. **Phase 的三層分解**:
   ```
   Phase_total = Phase_mono + Phase_physical + Phase_residual
                 (輸入)      (物理模型)      (神經網路)
   ```

2. **ILD 顯式建模**:
   ```
   Meta: 隱式在 warp 中
   我們: 顯式 magnitude 網路
   ```

3. **頻率自適應**:
   ```
   低頻: 物理主導
   中頻: 物理 + 學習
   高頻: 學習主導
   ```
