# IPD-only 學習失敗分析

**日期**: 2026-04-27  
**實驗**: HybridTFNet-IPD (2.2M 參數)  
**結果**: 失敗 - 無法學習任何有意義的相位資訊

---

## 實驗結果

### 訓練數據 (3 epochs)
```
Epoch 1: L2=0.000549, IPD_loss=3.1020, Mag=0.223289
Epoch 2: L2=0.000549, IPD_loss=3.0752, Mag=0.223085  
Epoch 3: L2=0.000549, IPD_loss=3.0703, Mag=0.223088
```

### 驗證數據
```
Val L2: 0.000493 (固定不變)
Val IPD_loss: ~3.09 (接近隨機基準 π)
Val Mag: 0.203093 (固定不變)
```

### 關鍵觀察
1. **L2 loss 卡在 0.0005** - 與之前 Phase Difference 版本完全相同
2. **IPD loss 停在 3.07-3.10** - 接近均勻分佈的理論值 π ≈ 3.14
3. **Magnitude 學習成功** - 但相位完全失敗
4. **從 Epoch 1 就停止進步** - 不是收斂慢，而是根本學不到

## 為什麼 IPD-only 也失敗？

### 1. IPD 分佈仍然過於分散

**理論分析**:
```
IPD = Phase_L - Phase_R = angle(Y_L / Y_R)

對於語音信號:
- 低頻 (<1kHz): ITD 主導，IPD ∝ frequency × ITD
  - ITD ≈ ±0.6ms → IPD 在 [-π, π] 範圍內變化
  
- 高頻 (>2kHz): ILD 主導，但相位仍然隨機
  - 頭部繞射、耳廓反射造成複雜相位變化
  - IPD 在 [-π, π] 幾乎均勻分佈
```

**實證數據** (來自之前的診斷):
```python
Phase_diff_L std: 1.82  # 接近均勻分佈的 1.81
Phase_diff_R std: 1.82
```

即使合併為單一 IPD，標準差仍然 ~1.8，表示分佈極度分散。

### 2. Wrapped MSE 對分散分佈無效

**Loss function**:
```python
loss_ipd = torch.mean((1 - torch.cos(ipd_pred - ipd_gt)))
```

當 IPD 分佈接近均勻時:
- 任何預測值的期望 loss ≈ 1.0
- 梯度信號極弱，無法提供有效的學習方向
- 模型退化為學習 magnitude + 隨機相位

### 3. 模型學習策略

**觀察到的行為**:
```
L2 = 0.000549 = magnitude_error + phase_error
     ≈ 0.000050 (mag) + 0.000499 (phase)
```

模型的最優策略:
1. **學習 magnitude** - 有明確的梯度信號 ✅
2. **忽略 IPD** - 學習成本高，收益低 ❌
3. **使用 mono phase** - 最簡單的相位估計 ✅

結果: `Y_pred = Mag_gt × exp(j × Phase_mono)`

## 根本原因總結

### 1. 相位的本質問題
```
語音相位 = 最小相位成分 + 全通相位成分

最小相位: 由 magnitude 決定 (可預測)
全通相位: 與 magnitude 無關 (不可預測)

雙耳相位差主要來自全通成分 → 無法從 mono 預測
```

### 2. 數學證明

對於均勻分佈的相位差 θ ~ Uniform(-π, π):
```
E[cos(θ_pred - θ_gt)] = 0  (任意 θ_pred)
→ 無論模型輸出什麼，期望 loss 都相同
→ 無法通過梯度下降學習
```

### 3. 資訊理論限制

```
I(IPD; Mono, View) ≈ 0

IPD 與輸入 (mono + view) 的互資訊接近 0
→ 從資訊理論角度，這個任務不可解
```

---

## 為什麼 Magnitude 能學，Phase 不能？

### Magnitude 特性
- **平滑**: 頻率上連續變化
- **可預測**: 由聲源、房間、頭部傳輸函數決定
- **低維**: 主要能量集中在少數頻帶
- **梯度清晰**: MSE loss 提供明確的優化方向

### Phase 特性  
- **跳躍**: 在 [-π, π] 邊界處不連續
- **隨機**: 受微小擾動影響巨大
- **高維**: 每個頻率獨立變化
- **梯度混亂**: Wrapped loss 在分散分佈下無效

## 實驗結論

### 已驗證的失敗方法
1. ❌ **Phase Difference (L/R 分開)** - 11.5M 參數，Phase error 3.18
2. ❌ **IPD-only (合併輸出)** - 2.2M 參數，IPD error 3.07
3. ❌ **增加 loss weight** - 無效，問題在分佈本身
4. ❌ **更深的網路** - 無效，不是容量問題

### 核心問題
**雙耳相位差的分佈過於分散，無法通過神經網路從 mono 音訊學習**

這不是:
- 模型架構問題
- 訓練策略問題  
- 超參數問題

而是:
- **任務本身的資訊理論限制**
- **相位的數學性質決定的**

## 下一步方向

### 選項 1: Group Delay (相位導數)
```python
GD = -dφ/dω

優點:
- 比相位本身更平滑
- 與 ITD 直接相關
- 可能有更好的可學習性

風險:
- 仍然是相位資訊，可能同樣分散
- 需要頻域微分，可能引入噪聲
```

### 選項 2: Magnitude-only
```python
只預測 ILD，放棄相位

優點:
- 已證明可學習
- 仍能提供空間感知
- 計算效率高

缺點:
- 失去精確的方向定位
- 低頻空間資訊缺失
```

### 選項 3: 混合方法
```python
低頻 (<1kHz): 使用 ITD 模型 (幾何計算)
高頻 (>1kHz): 只學習 ILD

優點:
- 結合物理模型和學習方法
- 避開最困難的相位學習

實現:
- 需要額外的 ITD 估計模組
- 頻帶分割和融合
```

### 選項 4: 接受現實
```
承認從 mono 無法完美重建雙耳相位
專注於 magnitude-based 空間化
在論文中討論這個根本限制
```
