## 核心發現

### 1. Phase 學習的根本限制

**數學證明**:
```
IPD 分佈: std ≈ 1.82 (接近均勻分佈 1.81)
互資訊: I(Phase; Mono, View) ≈ 0
→ Phase 無法從輸入預測
```

**實驗證據**:
- 所有架構都失敗
- 所有 loss weight 都無效
- 增加參數量無效
- 簡化輸出無效

**結論**: 這是**資訊理論限制**，不是工程問題

### 2. 為什麼「分兩塊」失敗？

**問題**:
1. Phase 根本不可學
2. 分支梯度衝突
3. 過度參數化 (8M 參數浪費)
4. 掩蓋了真正的問題

**詳細分析**: `實驗記錄/20260427_為什麼分兩塊失敗.md`

### 3. Magnitude vs Phase

| 特性 | Magnitude | Phase |
|------|-----------|-------|
| 分佈 | 平滑、連續 | 跳躍、分散 |
| 可預測性 | 高 | 極低 |
| 梯度信號 | 清晰 | 混亂 |
| 學習結果 | ✅ 成功 | ❌ 失敗 |

## 下一步計劃

### E5: Magnitude-only Baseline (今天)

**目標**: 確認 Magnitude 學習的上限

**架構**:
```python
class MagnitudeOnlyBaseline(nn.Module):
    def forward(self, mono_stft, view):
        # 只學習 ILD
        mag_L = self.mag_net_L(mono_stft, view)
        mag_R = self.mag_net_R(mono_stft, view)
        
        # Phase 直接用 mono
        phase = torch.angle(mono_stft)
        
        Y_L = mag_L * torch.exp(1j * phase)
        Y_R = torch.exp(1j * phase)
        
        return torch.stack([Y_L, Y_R], dim=1)
```

**預期**:
- L2 loss: ~0.0005 (只有 magnitude error)
- 空間感: 仍然存在 (ILD 提供)
- 定位精度: 降低 (缺少 ITD)

### E6: WaveformSpatializer (明天開始)

**目標**: 完全避開 Phase 問題

**架構**:
```python
class WaveformSpatializer(nn.Module):
    """時域直接學習，不經過 STFT"""
    
    def forward(self, mono_wave, view):
        # 學習時域 filter
        filter_L = self.filter_net_L(view)
        filter_R = self.filter_net_R(view)
        
        # 時域卷積
        binaural_L = F.conv1d(mono_wave, filter_L)
        binaural_R = F.conv1d(mono_wave, filter_R)
        
        return torch.stack([binaural_L, binaural_R], dim=1)
```

**創新點**:
1. 完全避開 STFT 和 Phase wrapping
2. 直接優化聽覺目標 (waveform MSE)
3. 可以學習非線性效應
4. 與 DPATFNet 完全不同

**技術挑戰**:
- 長序列建模 → dilated convolution
- 計算效率 → 可接受

**預期**:
- 可能突破 Phase 學習的限制
- 更好的聽覺質量
- 更高的創新性

## E8: HybridPhysical + Staged Training ❌

**時間**: 2026-04-27 16:48 - 17:10  
**狀態**: ❌ 失敗 (Early stopped)  
**目標**: 分階段訓練突破 Phase 學習

### 配置

**模型**: HybridPhysicalLearned (2.0M params)
- Physical ITD + Learned Magnitude + Learned Residual ITD

**訓練**: 3-stage (65 epochs total)
- Stage 1 (20 ep): Magnitude only, freeze residual_itd_net
- Stage 2 (30 ep): Add IPD, unfreeze residual_itd_net  
- Stage 3 (15 ep): End-to-end, early stopped

**對照**: Magnitude-only baseline (16 epochs, early stopped)

### 結果

| Metric | Staged | Magnitude-only | DPATFNet | 差距 |
|--------|--------|----------------|----------|------|
| **Waveform L2** | **0.000864** | **0.000857** | **0.000719** | +20% |
| IPD | 0.89 | 1.00 | 3.18 | - |
| Phase L | 0.9991 | 0.9991 | 0.9991 | - |
| Phase R | 0.9989 | 0.9992 | 0.9992 | - |

### 關鍵發現

1. **Staged ≈ Magnitude-only** (差 0.8%)
   - 分階段訓練沒有幫助
   - Phase 學習完全無效

2. **兩者都比 DPATFNet 差 20%**
   - 這 20% 來自 Phase
   - 但我們學不會

3. **IPD 改善但 L2 不改善**
   - Stage 2: IPD 1.00→0.89
   - 但 Waveform L2 不變
   - IPD 不是正確的監督信號

4. **Magnitude-only 快速收斂**
   - Epoch 1 就達到最佳
   - 這就是 magnitude 學習的極限

### 結論

**8 種方法全部失敗**:
- E1-E3: Phase Diff ❌
- E4: IPD ❌
- E6: Waveform ❌
- E7: HybridPhysical ❌
- E8: Staged Training ❌

**Phase 學習從 Monaural Audio 可能是不可能的**

## E9: Improved Residual Phase (計劃中)

**時間**: 2026-04-27 17:30 (預計)  
**狀態**: 🔄 準備中  
**目標**: 通過約束和 Perceptual Loss 改進 Residual Phase

### 動機

**E7/E8 失敗的原因**:
- Residual 網路太大，學習太多
- 沒有約束 residual 的大小
- Loss function 不夠好

**改進策略**:
1. ✅ 限制 residual 大小 (tanh + scale)
2. ✅ Residual regularization (懲罰大的 residual)
3. ✅ 頻率相關 mask (低頻大，高頻小)
4. ✅ Perceptual loss (multi-resolution STFT)

### 配置

**模型**: ImprovedResidualPhaseNet (2.0M params)
```python
# Physical ITD (固定)
physical_itd = compute_physical_itd(view)

# Learned residual (受約束)
residual = tanh(residual_net(...)) * 0.5  # 限制在 [-0.5, 0.5]
freq_mask = linspace(1.0, 0.1, 513)       # 低頻→高頻
residual = residual * freq_mask            # 高頻更小

# 合併
phase_L = mono_phase + (physical_itd + residual) / 2
phase_R = mono_phase - (physical_itd + residual) / 2
```

**Loss Function**:
```python
# 1. Waveform L2
waveform_loss = mse_loss(pred, target)

# 2. Perceptual loss (multi-resolution STFT)
perceptual_loss = multi_resolution_stft_loss(pred, target)

# 3. Residual regularization
residual_reg = mean(residual ** 2)

# 4. Temporal smoothness
temporal_smooth = mean((phase[:, 1:] - phase[:, :-1]) ** 2)

# Total
total_loss = (
    10.0 * waveform_loss +
    5.0 * perceptual_loss +
    0.01 * residual_reg +
    0.1 * temporal_smooth
)
```

**訓練**:
- Epochs: 100 (early stopping: 15)
- Batch size: 16
- Learning rate: 3e-4
- Optimizer: Adam

### 預期結果

**樂觀**: L2 < 0.00070 (超越 DPATFNet)  
**中等**: L2 ≈ 0.00075 (接近 DPATFNet)  
**悲觀**: L2 > 0.00080 (仍然失敗)

### 成功標準

**最低**:
- L2 < 0.00080 (比 Magnitude-only 好)
- Phase L/R < 0.99 (有學到一些)

**理想**:
- L2 < 0.00070 (持平或超越 DPATFNet)
- Phase L/R < 0.95 (明顯改善)

**突破**:
- L2 < 0.00065 (顯著超越 DPATFNet)
- Phase L/R < 0.90 (大幅改善)

## 經驗教訓

### 1. 不要盲目相信「最佳實踐」
- Magnitude-Phase 分離看起來合理
- 但實際上掩蓋了根本問題
- 要質疑假設，驗證可行性

### 2. 負面結果也是貢獻
- 證明什麼不可行同樣重要
- 系統性的失敗分析有價值
- 為未來研究指明方向

### 3. 簡單往往更好
- DPATFNet (515K) vs HybridTFNet (11.5M)
- 複雜架構沒有帶來優勢
- 先驗證可行性，再增加複雜度

### 4. 理論分析很重要
- 資訊理論視角揭示根本限制
- 數學分析指導實驗設計
- 不要只靠經驗和直覺

### 5. 改變問題定義
- 從「如何學習 Phase」
- 到「如何不需要學習 Phase」
- 創新往往來自重新定義問題
