# 多 Agent 討論：分階段訓練策略

**主題**：如何改進 HybridPhysicalLearned？  
**靈感**：DPATFNet 的分階段訓練  
**目標**：突破 L2 = 0.000719 的瓶頸

---

## Agent A (樂觀派 - ML 工程師)

### 分析

**DPATFNet 的成功經驗**：
```python
# Stage 1: 學習粗略的 magnitude + phase
loss_stage1 = MSE(pred_stft, target_stft)

# Stage 2: 精細化 ILD + IPD
loss_stage2 = MSE(pred_stft, target_stft) + λ_ILD × ILD_loss + λ_IPD × IPD_loss

# Stage 3: 端到端 L2 優化
loss_stage3 = MSE(pred_wave, target_wave)
```

**為什麼有效**：
1. 先學習整體結構
2. 再強化空間線索
3. 最後端到端微調

### 提議：三階段訓練

```python
# Stage 1: Magnitude + Physical ITD (20 epochs)
# 目標：學好 ILD，Phase 用物理模型
loss_1 = 1.0 × MSE(pred_mag, target_mag) + 0.1 × MSE(pred_wave, target_wave)

# 固定 residual_ITD = 0
# 只訓練 magnitude_net

# Stage 2: 加入 IPD 約束 (30 epochs)
# 目標：強化雙耳相位差
loss_2 = 0.5 × MSE(pred_mag, target_mag) 
       + 2.0 × IPD_loss
       + 0.5 × MSE(pred_wave, target_wave)

# 解凍 residual_itd_net
# 學習頻率相關的 ITD 修正

# Stage 3: 端到端 L2 優化 (50 epochs)
# 目標：直接優化聽覺目標
loss_3 = 10.0 × MSE(pred_wave, target_wave)
       + 1.0 × IPD_loss
       + 0.1 × MSE(pred_mag, target_mag)

# 所有參數一起訓練
# 重點在時域 L2
```

### 為什麼可能更好？

1. **漸進式學習**
   - 先學簡單的（magnitude）
   - 再學困難的（phase）
   - 避免一開始就陷入局部最優

2. **明確的監督信號**
   - Stage 1: Magnitude 有清晰梯度
   - Stage 2: IPD 有物理先驗
   - Stage 3: L2 端到端優化

3. **比 DPATFNet 更好的先驗**
   - 我們有物理 ITD
   - 我們有顯式 ILD
   - 我們有頻率自適應

### 預期效果

```
Stage 1 結束: L2 ≈ 0.0008, IPD ≈ 1.0
Stage 2 結束: L2 ≈ 0.0006, IPD ≈ 0.7
Stage 3 結束: L2 ≈ 0.0004, IPD ≈ 0.5  ← 突破 DPATFNet!
```

## Agent B (懷疑派 - 理論研究員)

### 質疑

**這個方案的問題**：

#### 1. 分階段訓練不能改變資訊理論限制

```
I(Phase; Mono, View) ≈ 0

無論分幾個階段：
- Stage 1, 2, 3, ...
- 資訊仍然不足
- Phase 仍然學不會
```

**類比**：
```
你不能通過「先學一點，再學一點」
來學會一個根本不存在的東西

就像：
- 先學習 1/3 的「永動機」
- 再學習 1/3 的「永動機」
- 最後學習 1/3 的「永動機」
→ 仍然造不出永動機
```

#### 2. DPATFNet 也沒有學好 Phase

```
DPATFNet 結果:
L2: 0.000719
Phase: 3.18 (完全隨機)

它的 L2 低是因為:
- Magnitude 學得好
- Phase 用了 mono phase
- 不是因為學會了 Phase
```

**證據**：
```python
# DPATFNet 的實際行為
pred_phase ≈ mono_phase + random_noise

# 不是
pred_phase ≈ target_phase
```

#### 3. 我們已經試過類似的方法

```
HybridTFNet-IPD:
- 先學 magnitude (frequency branch)
- 再學 IPD (time branch)
- 結果: Phase 仍然失敗

WaveformSpatializer:
- 時域端到端學習
- 加入 magnitude + IPD 約束
- 結果: Phase 仍然失敗

HybridPhysical:
- 物理 ITD + 學習 ILD
- 分階段訓練
- 結果: Phase 仍然失敗
```

### 反駁 Agent A

**「為什麼可能更好」的每一點都有問題**：

1. **漸進式學習**
   - 問題：Phase 不是「困難」，是「不可能」
   - 漸進式無法解決不可能的問題

2. **明確的監督信號**
   - Magnitude: 有監督 ✓
   - IPD: 有監督但資訊不足 ✗
   - L2: 被 magnitude 主導，Phase 信號弱 ✗

3. **比 DPATFNet 更好的先驗**
   - 物理 ITD: 只對低頻準確
   - 顯式 ILD: 這個確實好
   - 頻率自適應: 高頻仍然學不會

### 預測

```
Stage 1 結束: L2 ≈ 0.0008, IPD ≈ 1.0  ✓ 可能
Stage 2 結束: L2 ≈ 0.0007, IPD ≈ 0.9  ✓ 可能
Stage 3 結束: L2 ≈ 0.0007, IPD ≈ 0.85 ✗ 不會突破 0.0007
```

**原因**：
- L2 = 0.0007 是 magnitude-only 的極限
- 剩下的 0.0002 來自 Phase 誤差
- 這部分無法通過學習改善
