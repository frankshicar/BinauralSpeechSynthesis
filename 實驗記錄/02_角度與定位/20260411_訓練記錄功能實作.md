## 一、功能概述

訓練時會自動記錄每個 epoch 的詳細資訊，包括：
- 所有 loss 值（l2, phase, ipd, intermediate losses, accumulated loss）
- 學習率變化
- 每個 epoch 的訓練時間
- 時間戳記

記錄會即時保存為 JSON 和 CSV 格式，方便後續分析和視覺化。

## 五、記錄的資訊

### 每個 Epoch 記錄

- `epoch` - Epoch 編號
- `timestamp` - 時間戳記
- `learning_rate` - 當前學習率
- `epoch_time_seconds` - 訓練時間（秒）
- `epoch_time_formatted` - 訓練時間（格式化）
- `l2` - L2 loss
- `phase` - Phase loss
- `ipd` - IPD loss
- `intermediate_l2` - 中間層 L2 loss
- `intermediate_phase` - 中間層 Phase loss
- `intermediate_ipd` - 中間層 IPD loss
- `accumulated_loss` - 總 loss

### 訓練總結

- 總 epochs 數
- 總訓練時間
- 開始/結束時間
- 最終所有 loss 值
- 最佳 epoch 和對應的 loss
- Loss 改善幅度和百分比

## 七、實際應用範例

### 找出最佳 Checkpoint

```python
import json

with open('outputs/training_logs/training_history.json') as f:
    history = json.load(f)

best = min(history, key=lambda x: x['accumulated_loss'])
print(f"最佳 checkpoint: epoch-{best['epoch']}")
print(f"使用模型: outputs/binaural_network.newbob.net.epoch-{best['epoch']}")
```

### 比較不同訓練配置

```python
import json
import matplotlib.pyplot as plt

# 載入兩次訓練的記錄
with open('outputs_ipd_0.1/training_logs/training_history.json') as f:
    history1 = json.load(f)

with open('outputs_ipd_0.2/training_logs/training_history.json') as f:
    history2 = json.load(f)

# 比較 accumulated_loss
plt.plot([e['epoch'] for e in history1], 
         [e['accumulated_loss'] for e in history1], 
         label='IPD=0.1')
plt.plot([e['epoch'] for e in history2], 
         [e['accumulated_loss'] for e in history2], 
         label='IPD=0.2')
plt.legend()
plt.xlabel('Epoch')
plt.ylabel('Accumulated Loss')
plt.title('IPD Weight Comparison')
plt.savefig('ipd_comparison.png')
```

### 分析訓練穩定性

```python
import json
import numpy as np

with open('outputs/training_logs/training_history.json') as f:
    history = json.load(f)

losses = [e['accumulated_loss'] for e in history]
diffs = np.diff(losses)

print(f"平均改善: {np.mean(diffs):.6f}")
print(f"標準差: {np.std(diffs):.6f}")

# 找出異常波動
large_jumps = np.where(np.abs(diffs) > 0.01)[0]
if len(large_jumps) > 0:
    print(f"\n發現 {len(large_jumps)} 個大幅波動:")
    for idx in large_jumps:
        print(f"  Epoch {idx+1} -> {idx+2}: {diffs[idx]:+.6f}")
```

## 十一、故障排除

### 問題：找不到訓練記錄

確認訓練已經開始，並且至少完成了一個 epoch。記錄檔案會在第一個 epoch 完成後創建。

### 問題：視覺化腳本報錯

確認已安裝 matplotlib：
```bash
pip install matplotlib
```

### 問題：CSV 檔案亂碼

使用支援 UTF-8 的編輯器開啟，或在 Excel 中：
1. 資料 → 從文字/CSV
2. 檔案來源選擇 "65001: Unicode (UTF-8)"

### 問題：監控腳本沒有更新

確認訓練正在進行中，並且 `--log_dir` 路徑正確。
