## 📂 資料夾詳細說明

### 🔧 核心功能

#### `src/` - 核心原始碼模組
- `models.py` - 神經網路模型定義
- `trainer.py` - 訓練器（含 IPD Loss）
- `losses.py` - 損失函數（L2, Phase, IPD）
- `synthesis_utils.py` - 合成工具和角度校正
- `doa.py` - DOA 估計方法（ITD, ILD, Hybrid）
- `dataset.py` - 資料集處理
- `alignment.py` - 時間對齊
- `utils.py` - 工具函數
- 其他模組...

#### 根目錄核心腳本
- `train.py` - 主要訓練腳本
- `evaluate.py` - 模型評估腳本
- `synthesize.py` - 雙耳音訊合成
- `finetune.py` - 模型微調
- `monitor_training.py` - 即時監控訓練進度
- `visualize_training.py` - 訓練過程視覺化

### 🧪 實驗和測試

#### `experiments/` - 實驗性腳本
- **角度校正**：`calibrate_angle.py`, `build_angle_cache.py`
- **DOA 分析**：`compare_doa_methods.py`, `diagnose_ild.py`
- **診斷工具**：`check_90deg_itd.py`, `diagnose_itd_all_angles.py`
- **測試腳本**：`test_ipd_gradient.py`, `test_training_logs.py`
- **資料處理**：`align_dataset.py`, `count_trainset_angles.py`
- 其他實驗工具...

#### `scripts/` - Shell 腳本工具
- `start_training_with_ipd.sh` - 啟動 IPD 訓練
- `start_new_training.sh` - 開始新訓練
- `resume_training_example.sh` - 恢復訓練範例
- `verify_fix.sh` - 驗證修復腳本

## 🎯 使用指南

### 日常開發
```bash
# 訓練模型
python train.py --dataset_directory ./dataset/trainset --artifacts_directory ./outputs

# 評估模型  
python evaluate.py --dataset_directory ./dataset/testset --model_file ./outputs/binaural_network.newbob.net

# 合成音訊
python synthesize.py --input mono.wav --angle 60 --output binaural_60deg.wav
```

### 監控和分析
```bash
# 監控訓練
python monitor_training.py --log_dir ./outputs/training_logs

# 視覺化結果
python visualize_training.py --log_dir ./outputs/training_logs

# 診斷 ITD
python experiments/diagnose_itd_all_angles.py --model ./outputs/binaural_network.newbob.net
```

### 實驗和測試
```bash
# 角度校正
python experiments/calibrate_angle.py --input mono.wav --desired 60

# DOA 方法比較
python experiments/compare_doa_methods.py --input mono.wav

# 測試 IPD Loss
python experiments/test_ipd_gradient.py
```
