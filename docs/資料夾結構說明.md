# 專案資料夾結構說明

整理日期：2026-05-11

## 主要入口

```text
BinauralSpeechSynthesis/
├── src/                         # 核心 Python 模組
├── training/                    # 主要訓練程式
├── scripts/                     # 啟動腳本與 active 診斷工具
├── tests/                       # 自動測試
├── dataset/                     # 訓練/測試資料
├── docs/                        # 技術文件
├── 實驗記錄/                    # 實驗紀錄與結果整理
├── experiments/                 # 非 canonical 的一次性實驗工具
├── archive/                     # 舊 run、舊輸出、原始備份
├── paper/                       # 論文 PDF 與 manuscript notes
├── geowarp_film_v6_4/           # 目前 active v6.4 訓練輸出
├── geowarp_film/                # 舊 GeoWarpFiLM 綜合評估輸出
├── geowarp_film_v6_release/     # v6 release snapshot
├── binaural_network_3blocks.net # Meta pretrained 3-block model
├── README.md
└── evaluate*.py                 # 評估入口
```

## Active Code

- `src/`：模型、loss、dataset、warping、utility 等核心模組。
- `training/`：主要訓練程式。目前 v6.4 是
  `training/train_geowarp_film_v6_4.py`。
- `scripts/`：訓練啟動腳本與 active 診斷工具。目前 v6.4 launcher 是
  `scripts/start_train_geowarp_film_v6_4.sh`。
- repo root 的 `evaluate*.py`：評估入口，保留在根目錄方便執行。

## Data And Outputs

- `dataset/`：資料集。`testset` 是 Meta test set；`test_static` 是自建
  static test set。
- `binaural_network_3blocks.net`：Meta pretrained model。
- `geowarp_film_v6_4/`：目前正在使用的 v6.4 run 目錄。訓練期間不要搬動。
- `geowarp_film/`：舊版綜合評估輸出，許多比較紀錄仍引用它。
- `archive/legacy_runs/`：已歸檔的 v4 到 v6.3 run 目錄。
- `archive/legacy_outputs/`：舊生成音訊與 evaluation output。

## Docs And Records

- `docs/architecture/`：模型與方法設計文件，依研究線分成
  `geowarp_film/`、`dpatfnet/`、`pgcn/`、`proposals/`、`infrastructure/`。
- `docs/training/`：訓練操作指南。
- `docs/analysis/`：整理、清理、分析報告。
- `docs/research/`：研究討論與 agent notes。
- `docs/reports/`：獨立報告與 planning notes。
- `docs/presentations/`：簡報。
- `實驗記錄/`：正式實驗紀錄。新增實驗請放到對應主題資料夾，並更新
  `實驗總表.md`。

## Experiments

`experiments/` 只放非 canonical、可能需要更新 import/path 才能跑的工具：

- `experiments/diagnostics/`：ITD/ILD、DOA、checkpoint、模型行為診斷。
- `experiments/data_tools/`：角度、position、dataset 對齊與生成工具。
- `experiments/training/`：舊 quick training prototypes。
- `experiments/audio_tools/`：音訊增強實驗。

## 新增檔案放置規則

- 新架構與方法說明：`docs/architecture/`
- 訓練操作與 resume 說明：`docs/training/`
- 每次重要 run 的紀錄：`實驗記錄/`
- 一次性分析腳本：`experiments/diagnostics/`
- 資料產生/整理工具：`experiments/data_tools/`
- 正式訓練程式：`training/`
- 正式啟動腳本：`scripts/`
- 舊輸出與不再 active 的 run：`archive/`
