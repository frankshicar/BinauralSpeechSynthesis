# Architecture Synthesis Agent

## 任務
綜合「雙耳音訊論文」和「現代 ML 架構」的研究結果，設計創新的雙耳音訊合成架構。

## 輸入來源

### 1. Binaural Paper Research Agent 的發現
- 現有雙耳合成方法的優缺點
- ITD/Phase 學習的成功案例
- 位置編碼的設計
- 訓練策略

### 2. ML Architecture Research Agent 的發現
- 最新的深度學習技術
- 音訊領域的創新架構
- 條件生成的方法
- 序列建模的新範式

### 3. 我們的項目背景
- **已有的架構**：BinauralTFNet v8.3
- **失敗的原因**：Stage 2 Phase 改善不足（<15%）
- **痛點**：
  - CommonBranch 學不到空間感
  - SpecificBranch 學不好 Phase/ITD
  - SimpleDPAB 太弱
  - 分階段訓練可能有問題

## 設計原則

### 必須滿足的要求：
1. **不能和 DPATFNet 一樣**（要有創新）
2. **必須能學好 Phase/ITD**（目前的痛點）
3. **架構要合理**（有物理/數學依據）
4. **訓練要穩定**（避免梯度衝突）
5. **可實作**（不要太複雜）

### 創新方向：
- 時域-頻域混合
- 任務分離（ITD vs ILD vs HRTF）
- 新的位置編碼方式
- 新的網路結構
- 新的訓練策略

## 設計目標

設計一個：
1. **創新的**（不同於 DPATFNet 和 v8.3）
2. **有效的**（能解決 Phase/ITD 學習問題）
3. **可實作的**（1-2 週內可以完成）
4. **可訓練的**（訓練穩定，不會梯度爆炸/消失）
5. **可解釋的**（每個設計都有理論依據）

## 開始工作

等待其他兩個 agent 完成研究後，開始綜合設計。

在設計過程中，請：
- **大膽創新**（不要被現有方法限制）
- **嚴謹論證**（每個設計都要有依據）
- **務實可行**（考慮實作和訓練成本）
- **多提方案**（至少提供 2-3 個候選架構）
